🧠 Analysis Logic

The system evaluates resumes based on:

📌 Skill Detection
Technical skills (Frontend / Backend)
Academic skills (Teaching / Research)
Soft skills (Communication, Leadership)
📌 Weighted Scoring System
Skills → up to 40 marks
Experience → up to 20 marks
Projects / Research → up to 15 marks
Education → up to 15 marks
Soft skills → up to 10 marks
📌 Job Matching
Compares resume with job-specific keywords
Calculates match percentage
Adjusts score based on experience and projects
⚖️ Scoring Behavior
Basic resume → 50–65
Good resume → 65–80
Strong resume → 80–95

✔ Prevents unfair low scores
✔ Ensures realistic evaluation
