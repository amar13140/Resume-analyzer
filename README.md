🤖 AI Resume Analyzer (Smart & Balanced Version)

An intelligent web-based AI Resume Analyzer built using HTML, CSS, and JavaScript that evaluates resumes, detects their type, and determines how well they match a specific job role with a balanced and realistic scoring system.
