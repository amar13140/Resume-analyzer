# 🤖 AI Resume Analyzer

A simple web-based AI Resume Analyzer built using HTML, CSS, and JavaScript.

## 🚀 Features
- Resume scoring system 📊
- Skill detection 🧠
- Strength & weakness analysis ✅❌
- Improvement suggestions 💡

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript

## 🎮 How to Use
1. Paste your resume text
2. Click "Analyze Resume"
3. View your score and feedback
